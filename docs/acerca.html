Acerca del Proyecto

El proyecto nace a partir de un problema que muchas personas vivimos todos los días: el mal estado de las calles en el barrio San cristobal. Es muy común encontrar huecos, grietas o desniveles que hacen difícil caminar, andar en bicicleta o manejar. Estos daños en la vía no solo molestan, sino que también pueden ser peligrosos para quienes viven o transitan por allí. A partir de esta situación, surgió la idea de crear un proyecto que nos ayude a entender mejor lo que está pasando. Aunque por ahora no se usaron herramientas muy complejas ni datos técnicos, lo importante es empezar a mirar el problema con otros ojos y buscar formas de analizarlo con ayuda de la tecnología.

Este trabajo es un primer paso para aprender cómo usar páginas web y herramientas digitales que en el futuro pueden servir para identificar y mostrar los lugares más afectados del barrio. La idea es que, con el tiempo, este proyecto pueda crecer y convertirse en una forma útil de dar a conocer la situación y pensar en posibles soluciones.
