Tecnologías Utilizadas

Para este proyecto se utilizan diversas herramientas tecnológicas que permiten obtener, procesar y visualizar los datos geoespaciales. A continuación se detallan las principales tecnologías:

  • Leaflet.js: Biblioteca JavaScript para crear mapas interactivos.
  • OpenStreetMap: Plataforma de mapas libre y editable utilizada como base para el mapa.
  • GeoJSON: Formato para representar datos geoespaciales en JSON.

Además, se están explorando tecnologías como sensores IoT para mejorar la precisión en la identificación de huecos.
