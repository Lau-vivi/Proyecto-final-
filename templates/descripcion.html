Para el desarrollo de esta plataforma se emplearon diversas tecnologías y herramientas que permiten crear un sitio web funcional, interactivo y fácil de usar, facilitando el registro y visualización de los huecos en el barrio San Cristóbal.

HTML: Fue utilizado para estructurar el contenido de la página web, organizando los diferentes elementos como textos, imágenes y formularios de manera semántica y accesible.

CSS: Se encargó de dar estilo y formato a la página, definiendo colores, fuentes, márgenes y disposiciones que permiten una presentación visual coherente y atractiva. También se aplicaron técnicas de diseño responsivo para asegurar que el sitio se adapte correctamente a distintos tamaños de pantalla.

JavaScript: Para agregar interactividad y mejorar la experiencia del usuario, se utilizó JavaScript, que permitió validar los datos ingresados en los formularios y manejar eventos dinámicos dentro del sitio.

Leaflet.js: Para la representación cartográfica y la visualización de los puntos reportados, se integró Leaflet.js, una librería especializada en mapas interactivos que facilita la ubicación geográfica de los huecos sobre un mapa del barrio.

Python y Flask: En el backend, se usó el lenguaje Python junto con el framework Flask, que permite manejar las solicitudes del usuario, procesar la información enviada desde el formulario y conectarse con la base de datos para almacenar los registros.

SQLite: La base de datos utilizada fue SQLite, un sistema de gestión de bases de datos ligero que guarda los datos de forma estructurada y permite su posterior consulta y análisis.

Imágenes: El proyecto incluye una carpeta dedicada a imágenes, que contiene recursos gráficos como íconos, ilustraciones o capturas que ayudan a mejorar la presentación visual de la página. Estas imágenes también se utilizan como apoyo visual en el formulario o para representar elementos en el mapa.

Visual Studio Code: El proyecto fue desarrollado utilizando Visual Studio Code como editor de código, que ofrece herramientas para facilitar la escritura y organización del código fuente.

Git y GitHub: Para el control de versiones y gestión del proyecto, se empleó Git junto con la plataforma GitHub, lo que permitió mantener un historial ordenado de cambios y facilitar la colaboración y respaldo del código.
