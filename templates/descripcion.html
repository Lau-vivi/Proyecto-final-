Descripción de las Tecnologías Utilizadas

Este proyecto web fue creado utilizando HTML, hojas de estilo CSS y un archivo de tipo GeoJSON en formato `.js` para representar un polígono interactivo en el mapa. Cada una de estas herramientas cumple una función específica en la estructura y funcionalidad del sitio.

Páginas HTML

Se diseñaron cuatro páginas HTML principales: index.html como portada del sitio, acerca.html para presentar el objetivo del proyecto, descripcion.html para explicar las tecnologías utilizadas, y autor.html con información sobre la persona que desarrolló el proyecto. Estas páginas están interconectadas mediante una barra de navegación común.

Hojas de Estilo CSS

El diseño del sitio está controlado por dos archivos CSS: estilo_general.css, que define la apariencia general como colores suaves, tipografía, distribución de secciones y adaptabilidad del contenido; y estilos1.css, que se encarga de organizar visualmente elementos como los mapas y el contenido en columnas, por ejemplo, texto a la izquierda y el mapa a la derecha.

Carpeta de Imágenes

El proyecto incluye una carpeta de imágenes, donde se almacenan elementos visuales como la fotografía de la autora en formato JPEG. Estas imágenes se utilizan en las distintas secciones del sitio para complementar el contenido.
